Adding Photos

Adding Photos from the Web

If you use a website other than Flickr or Photobucket to store your photos, you can still add them to your project or stash pages.

Pulling images from the web is known as slurping. Please only do this if you have permission from the host.

  1. From the project or stash page you want to add photos to, click Add Photos.
  2. Click Slurp from Web.
  3. In the field, type or paste the complete URL for your photo and click Slurp It!.
  4. Note: If you aren't the copyright owner of the photo you're using, please make sure you have permission to use it.
    To change the copyright notice below your photo, just click on it and type in the correct information.
